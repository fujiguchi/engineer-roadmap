## 30×30 アプリ

1日30分×30日プランの単体アプリ（`app/index.html`、依存ゼロの単一ファイルSPA）。

- 今日のタスク自動表示（順序ベース — 遅れてもプランが崩れない）
- 30分カウントダウンタイマー
- 進捗リング・連続日数・30日マップ・日別メモ
- データはブラウザのlocalStorageに保存（端末ごと）

スマホでの利用推奨。ブラウザで開いて「ホーム画面に追加」するとアプリ感覚で使える。

## GitHub Pagesでの公開

Settings → Pages → Branch: `main` / root を選択して保存。数分後に
`https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される。
